# v2.2.4

- 修复日志视图默认主题，浅色模式改为浅底深字，深色模式改为深底浅字；同时移除固定深色渐变对浅色模式的干扰，日志背景预览也改为跟随主题。
- 统一 cron 规则在校验、预览和真实调度中的解析逻辑，修复多时间点与 Quartz 风格表达式在不同链路中表现不一致的问题。
- 订阅仓库新增 Access Token 鉴权支持，可在“无鉴权 / SSH 密钥 / Access Token”之间切换；Token 不回显、不写入 Git 远端地址，拉取时通过临时 HTTP 认证头注入。
- 修复 Windows 原生环境下脚本路径包含中文或空格时被错误判定为非法路径的问题。
- 修复 Windows 下 Python 执行优先命中 WindowsApps `python3.exe` 代理壳导致的启动失败，改为优先使用真实 `python.exe` / venv 解释器。
- 修复 Windows 下 Shell 脚本依赖 `bash` 必须在 PATH 中的问题，自动探测 Git Bash 常见安装目录。
- 修复 Node.js 脚本执行时 `sendNotify.js` preload helper 路径错误导致的 `MODULE_NOT_FOUND` 问题，统一改为绝对路径。
- 为 Python 运行环境补充 UTF-8 输出设置，改善 Windows 下中文路径与中文日志的可读性。
